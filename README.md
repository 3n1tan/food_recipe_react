# Project title

Food Recipe Application
This is a recipe-app web application. At default, it consists of 9 recipes including pictures, country, author and preparation steps. There is also an option for users to update the recipe database by insering their own recipe.

## Technologies used

Built with:

- REACT (npm run dev)
- JS
- CSS

## Required Dependencies
- vite (run npm install vite@latest)
- axios
- json-server (npx json-server --watch data/db.json --port 8000)
- react-dom, react-router
